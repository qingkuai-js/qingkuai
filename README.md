QingKuai (轻快) is a compiler-based frontend framework for building web interfaces. It transforms `.qk` source files into minimal, strictly optimized JavaScript.

Learn more at [qingkuai.dev](https://qingkuai.dev), or try it out in the [Playground](https://try.qingkuai.dev).
